How the Morph Effect Works

1
Aspect Ratio Container

The image container uses padding-top: 100% to maintain a square aspect ratio that scales with the card width.

2
Layered Images

Two absolutely positioned images are stacked in the same container with different opacity and scale values.

3
Morph Transition

On hover, we transition both images simultaneously with a custom cubic-bezier timing function:

.card:hover .primary { opacity: 0; transform: scale(0.95); } .card:hover .hover { opacity: 1; transform: scale(1); }
4
Performance Optimization

Using will-change, backface-visibility, and translateZ(0) ensures smooth animations.
